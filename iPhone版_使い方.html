農薬簡易検索

iPhone版 かんたん使い方ガイド

このアプリは パソコン・ケーブル不要。iPhoneの Safari だけでインストールできます（所要1分）。

📱 アプリのアドレスはこれ1つだけこれを開く

https://hirschmannielladiversa-bot.github.io/browser-pesticide-search/

見分け方：「github.io」（アイオー）が付いている方がアプリです。
「github.com」（コム）はGitHubの開発ページなので、開いてもアプリは出ません。

1
アプリを開く必ず Safari で開いてください（Chrome は不可）

下のアドレスを Safari で開きます。タップで全選択 → コピーできます。

https://hirschmannielladiversa-bot.github.io/browser-pesticide-search/
2
ホーム画面に追加これでアプリになります
  1. 画面の下中央にある 共有ボタン（四角に上向き矢印）をタップ
  2. メニューを下にスクロールして 「ホーム画面に追加」 をタップ
  3. 右上の 「追加」 をタップ
  4. ホーム画面に 「農薬検索」 アイコンが追加されます 🎉
ポイント： 以降はこのアイコンから起動すれば、全画面でサクサク動きます。電波が無い場所（ほ場・倉庫など）でもオフラインで検索できます。
3
検索して使うふだんの使い方
  1. アイコンをタップしてアプリを起動
  2. 作物名・農薬名・有効成分・RACコードなどを入力して検索
  3. 結果から登録内容（適用作物・希釈倍数など）を確認
4
データを最新にする登録変更があったとき

データは 毎週月曜 朝3時 に自動更新されます。それを待たずに 自分のタイミングで手動更新 することもできます。

  1. 下のアドレスを開く（GitHubのログインが必要な場合あり）
  2. 右側の Run workflow ボタンをタップ
  3. branch は main のまま、もう一度 Run workflow
  4. 約1分で完了。次にアプリをオンラインで開くと自動で最新化されます

⚠️ これは「更新操作用」のページです（アプリではありません）

https://github.com/hirschmannielladiversa-bot/browser-pesticide-search/actions/workflows/update-data.yml
この github.com のページは更新ボタンを押すための場所で、検索アプリは出ません。検索したいときは、いつものホーム画面のアイコン（または上の「github.io」のアドレス）から開いてください。
更新時は検索ロジックの自動テスト（21項目）が走り、1件でも失敗すると公開されない仕組みです。壊れたデータが配信される心配はありません。

これって何アプリ？USB接続が要らない理由

「ホーム画面に追加」で入るのは PWA という仕組みのアプリです。App Store も Xcode もケーブルも不要で、普通のアプリと同じように使えます。

7日で失効なしあり

こまったとき

「ホーム画面に追加」が見当たらない

Safari 以外（Chrome等）で開いている可能性があります。必ず Safari で開いてください。

データが古い気がする

電波のある場所でアプリを一度閉じて開き直すと最新化されます。すぐ反映したいときは「4. データを最新にする」を実行。

圏外でも使える？

はい。一度開いていれば、オフラインでも検索できます（その時点の保存データで動作）。

アイコンを消したい

他のアプリと同じく、ホーム画面でアイコンを長押し →「App を削除」。
