農薬検索ツール — 使い方

FAMIC 公開データと CropLife Japan の RAC コード表を統合したオフライン検索ツールです。 商品名・成分名・RAC コード・作物名・病害虫名から双方向に引けます。

1. 起動方法

ご利用前に毎回「データ更新」を実行することを推奨します。

農薬登録情報は月初に FAMIC で更新されるため、古いデータのままだと新規登録剤や登録変更が反映されません。検索前に データ更新.bat（Windows）または データ更新.command（macOS）を一度ダブルクリックしてください（所要 1〜2 分、ネットワーク接続が必要）。詳細は 6. データ更新。

1 まず データ更新.bat（Win）/ データ更新.command（Mac）をダブルクリック
2 完了したら build/これをクリック.html をダブルクリック

既定のブラウザで開きます。約 25 MB、初回読込は 1〜3 秒。起動後はネットワーク接続不要、検索内容は外部に送信されません。

データ更新済みの場合はステップ 2 だけで OK。 同日に再度開く場合は更新不要です。

OS操作
Windowsエクスプローラで build\これをクリック.html をダブルクリック
macOSFinder で build/これをクリック.html をダブルクリック

2. 検索のしかた

スペースで区切ると すべて満たす剤（AND 条件）に絞り込めます。

イミダクロプリド成分「イミダクロプリド」を含む全商品
アドマイヤー商品名に「アドマイヤー」を含む剤

/ キーで検索ボックスにフォーカスします。

3. サイドバーで絞込

検索バーの絞込に加えて、左サイドバーの条件が すべて AND で働きます。

項目動作
用途殺虫剤 / 殺菌剤 / 除草剤 のチェックボックス。複数可
登録状態有効 / 失効 のチェックボックス。どちらも既定で ON
除外「家庭向けを除外」（既定 ON） / 「混合剤のみ」
剤型水和剤 / 粒剤 / フロアブル剤 / 水溶剤 / 粉剤 / ジャンボ剤 / ベイト剤 / その他 から選択
作物入力即時絞込。部分一致、ひらがな/カタカナ自動変換。1,093 種の候補
対象病害虫・雑草入力即時絞込。部分一致、1,328 種の候補

入力欄のサジェスト: 作物名/病害虫名の欄は頻度順 Top 1,500 までオートコンプリート候補を表示します。

作物/病害虫を入力すると失効剤は自動で隠れます。 失効剤には適用情報（作物・病害虫）が無いためです。失効剤も見るには、これらの欄を空にしてください。

4. RAC バッジの見方

成分名の右隣に以下のいずれかのバッジが付きます。

バッジ意味例
4A M7 CropLife Japan 公式の RAC コード イミダクロプリド (4A) / ジネブ (M3)
対象外 RAC 体系の対象外成分。マウスオーバーで理由表示 脂肪酸エステル (物理作用) / 燐酸第二鉄 (軟体動物剤) / 微生物製剤
古典未分類 1960–90 年代登録で RAC コード未付与の古典剤 ダイムロン / ACN / DBEDC / ジエノクロル
新規未収録 2025–2026 新規登録で CropLife 表に未収録（次回更新で埋まる見込み） シクロピラニル / シンメチリン / ジンプロピリダズ

全 3,490 現行剤の 全成分が分類済み (カバレッジ 100%) です。

5. エクスポート

検索結果を 4 形式でダウンロードできます。

  • TSV: Excel にそのまま貼り付け可
  • CSV: 文字コード UTF-8 (Excel で開く場合は「データ」→「テキストファイル」から UTF-8 指定で読み込み)
  • Markdown: 報告書・メモ貼り付け用
  • JSON: プログラムから再利用

6. データ更新

FAMIC は月初に、CropLife RAC 表は 2〜3 ヶ月ごとに更新されます。

1 Windows: データ更新.bat / macOS: データ更新.command をダブルクリック

1〜2 分で最新データを取得し、これをクリック.html を再ビルドします。

前提詳細
Python 3.8 以上 macOS: 標準搭載（無ければ xcode-select --install）
Windows: python.org か Microsoft Store の "Python 3"。インストール時に "Add Python to PATH" にチェック
ネットワーク接続更新時のみ必要（検索時は不要）
openpyxl初回起動時に自動インストール

データ更新後はブラウザのキャッシュを再読み込みしてください。
Windows: Ctrl + F5 / macOS: Cmd + Shift + R

7. よくある質問・トラブル

Q. 作物名で検索したら失効剤が出なくなった

失効剤には適用情報（対象作物・病害虫）のデータが存在しないため、作物/病害虫で絞ると自動的に除外されます。失効剤も見たい場合は、作物/病害虫フィルタを空にしてください。

Q. エンドリン で検索したら「剤型: 乳剤」と出る、化合物名が出ない

剤型欄は 乳剤 / 粉剤 / 水和剤 / 粒剤 / フロアブル剤 / 水溶剤 / ジャンボ剤 / ベイト剤 / その他 の 9 分類に正規化されています。エンドリン等の化合物名は商品名または成分名の欄で確認できます。

Q. RAC コードが付いていない成分がある

RAC 未分類成分は理由バッジを表示します。上記「RAC バッジの見方」を参照してください。

Q. Windows で開くと文字化けする

HTML 自体は UTF-8 + <meta charset="UTF-8"> 宣言付きなので、どのブラウザでも文字化けしません。ファイル名（「これをクリック.html」「使い方.html」等の日本語名）が化ける場合は、配布時の ZIP 解凍ツールの問題です。対処は配布元に相談してください。

Q. データ更新.bat をダブルクリックすると一瞬でウィンドウが閉じる

Python 3 が入っていません。py -3 --version をコマンドプロンプトで実行して確認し、未インストールなら python.org からインストールしてください。

Q. 検索結果が多すぎる / 少なすぎる

スペース区切りで複数条件を AND で重ねられます。なす アブラムシ イミダクロプリド のように複数指定すると強力に絞り込めます。

農薬登録情報は FAMIC 公開 CSV および CropLife Japan RAC コード表 に基づきます。 データ著作権は各提供元に帰属します。
